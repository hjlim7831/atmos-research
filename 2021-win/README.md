### 폴더별 설명

- `2D_wrf` : 2차원 WRF 모델 모의 결과 분석용 폴더
    - `analysis` : 분석/시각화 코드
    - `model` : 모델 모의 관련 코드
- `OLD` : 3차원 WRF 모델 모의 결과 분석용 폴더
    - `analysis` : 분석/시각화 코드
    - `model` : 모델 모의 관련 코드
- `WRF-CMAQ` : WRF-CMAQ 모델 모의 결과 분석용 폴더
    - `analysis` : 분석/시각화 코드
    - `csv` : 모델 모의 결과의 유효성 검사를 위한 관측값 csv 파일
